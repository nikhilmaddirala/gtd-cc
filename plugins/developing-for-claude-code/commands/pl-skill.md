# Create new skill

## Overview

This command guides you through creating a new foundational skill using the advanced skill-creator methodology adapted for gtd-cc architectural patterns. CRITICAL: You MUST use the plugin-development-gtd skill for this task.

## Context

User will provide skill name, domain expertise area, and target plugin (if applicable). The command validates skill naming conventions and ensures proper integration with existing plugin structure or marketplace.

## Process

Load plugin-development-gtd skill first. Interactively guide the user through this skill and create-new-skill workflow step by step, collecting inputs and executing each step with confirmation.
