# Monorepo Git Skills

This plugin contains one skill with multiple sub-skills for different git operations.

## Skill overview

### monorepo-git

Main skill for all git operations in a monorepo with subtree support.

Sub-skills:
- `commit` - Analyze changes and create logical commits
- `push` - Push to monorepo remote only
- `status` - Quick working tree status
- `publish` - Curated subtree publishing with interactive selection
- `subtree-status` - Show divergence for all subtrees
- `subtree-manage` - CRUD operations for subtrees
- `graduate` - Move project from lab to production
