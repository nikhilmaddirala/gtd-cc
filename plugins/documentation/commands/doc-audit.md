# Documentation Audit

## Overview

This command guides users through docs-management - update.md interactively. CRITICAL: You MUST use docs-management for this task.

## Context

User will provide optional scope and focus areas. This command should parse the input and guide users through documentation health audit and systematic maintenance using scope=all, type=minor parameters.

## Process

- Load docs-management and its update.md workflow; you MUST do this first.
- Parse the user input and understand how it relates to docs-management/update.md with audit focus.
- Interactively guide the user through docs-management/update.md step by step, collecting inputs and executing each step with confirmation.
