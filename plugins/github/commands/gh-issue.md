# Issue Creation

## Overview

This command guides you through creating well-structured GitHub issues using the gh-manage skill. CRITICAL: You MUST use the gh-manage skill for this task.

## Context

User provides their initial request via $ARGUMENTS describing what needs to be done.

## Process

Load the gh-manage skill first. Follow its `issue-creation` workflow exactly as written to capture the user's request as a lightweight GitHub issue with clear problem statements, acceptance criteria, and appropriate labels.
