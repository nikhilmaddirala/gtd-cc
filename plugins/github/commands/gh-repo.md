# Repository Setup

## Overview

This command guides you through initial repository setup for issue-driven development using the gh-manage skill. CRITICAL: You MUST use the gh-manage skill for this task.

## Context

This is a one-time setup command for a new repository. No arguments required.

## Process

Load the gh-manage skill first. Follow its `repo-setup` workflow exactly as written to configure workflow labels, issue templates, PR templates, branch protection rules, and worktree directory structure.
