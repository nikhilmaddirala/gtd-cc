# Git Commits

## Overview

This command guides you through creating well-structured git commits using the gh-manage skill. CRITICAL: You MUST use the gh-manage skill for this task.

## Context

User has staged changes ready to commit. Changes will be provided as context when invoking this command.

## Process

Load the gh-manage skill first. Follow its `commit` workflow exactly as written to create a well-structured commit with conventional commit format (`<type>(<scope>): <description>`), clear intent, and proper message structure.
